# Tips & troubleshooting

- **My calendar color reset** — colors are stored using an
  Apple-compatible mechanism; if you use multiple calendar apps, the
  most recently saved color from *any* of them wins.
- **A calendar/contact I deleted disappeared instead of going to
  Trash** — very old items or ones purged past the retention window are
  gone permanently; check with your administrator about the trash
  retention policy.
- **My CalDAV/CardDAV app won't connect** — double-check you're using
  the **generated app-password username**, not your email address, as
  the login name.
- **I don't see a password field on my profile** — your account is
  managed by single sign-on; password changes happen with your
  organization's identity provider instead.
- **An event I was invited to isn't showing up** — if you're a local
  user, check the **Shared events** section at the bottom of the
  calendar sidebar; if you're external to the server, check your email
  for the invitation instead.
