# Sharing with other users or groups

Every calendar, address book, individual event, individual contact,
group, and (if you're an admin) the users/groups management pages have
an **Access control** panel. To share:

1. Open the resource's edit page and find **Access control**.
2. Enter another user's username or a group's slug.
3. Pick a permission level:
   - **Full control** — can do anything, including managing who else
     has access.
   - **Read** — can view but not change anything.
   - **Write** — can view and edit.
   - **Manage access** — can grant/revoke access to others, without
     necessarily being able to edit content.

You can share a whole calendar/address book, or just **one event or
contact** (open that item's own access-control page) if you don't want
to expose the rest of the collection. Some access entries are
system-managed (shown with a lock icon) and can't be revoked from the
UI — these exist to guarantee you always retain access to your own
data.

You can only grant access on a resource if you yourself have "Manage
access" (or "Full control") on it.
