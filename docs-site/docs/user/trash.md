# Trash

Deleting a calendar, address book, event, or contact doesn't erase it
immediately — it moves to **Trash**, which lists:

- Deleted calendars/address books
- Deleted individual events/contacts (even if their parent calendar is
  still active)

Each item can be **restored** or **permanently deleted**. Trash is
private to you — there's no time limit enforced by the UI itself, but
your server may be configured to automatically purge old trash after a
retention period, so don't rely on it as permanent storage. Once
something is permanently deleted (by you or by that automatic cleanup),
it's gone for good.
