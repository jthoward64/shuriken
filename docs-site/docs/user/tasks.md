# Tasks (to-dos)

Tasks live inside your calendar collections, so you pick which
calendar a task belongs to just like an event. From **Tasks**:

- Browse a paginated list, sorted by completion state then due date;
  overdue tasks are flagged.
- Each task shows a status (**Needs action / In process / Completed /
  Cancelled**) and priority (**High / Medium / Low / None**).
- Tasks support the same recurrence options as events.
- Create, edit, complete, or delete a task from the list.
- Toggle **show completed** to hide/show finished tasks.
